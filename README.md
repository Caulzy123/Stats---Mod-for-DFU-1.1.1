# StatsPlus-Mod-for-DFU-1.1.1
## Goals
I am bundling DaggerStats and AffiliationsPlus by Rith Essa into a single project that is completely optimised for all OS and devices. These are 2 great mods that must stay alive! And I figured because of their similar functions that it is efficient to bring them together and remove the .dll/Harmony.dfmod dependancy.

### What I have done so far:
1. NO MORE HARMONY
2. *DaggerStats* and *AffiliationsPLus* are now one mod called *StatsPlus*
3. Dedicated keybinds for both mod windows. Only while on the character sheet: "x" = Daggerstats, "Shift+A" = AffiliationsPlus.
4. Kept the vanilla Affiliations expanded window available by clicking "Affiliation". This offer simpler and easier to read data when required.
