# StatsPlus-Mod-for-DFU-1.1.1
## Goals
I am bundling DaggerStats and AffiliationsPlus by Rith Essa into a single project that is completely optimised for all OS and devices. These are 2 great mods that must stay alive! And I figured because of their similar functions that it is efficient to bring them together and remove the .dll/Harmony.dfmod dependancy.

### What I have done so far:
1. I have begun removing Harmony dependancy for AffiliationsPlus scripts by binding the Affiliations menu to the "Shift+a" key while on the character sheet, this mirrors the design for DaggerStats directly anyway, it also makes the mod more lean.
2. Changed the bug logs to a unified "StatsPlus" message for all, but respected the original directives and hooks to ensure they still are fundimentally the same under the hood.
